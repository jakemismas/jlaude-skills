# Strategic Plan Architect

## Purpose
Create comprehensive implementation plans for large features.

## Activation
Use before starting any significant development work.

## Process
1. Gather requirements from user
2. Analyze existing codebase for patterns
3. Identify affected objects, fields, and components
4. Create phased implementation plan
5. Identify risks and mitigations
6. Generate dev docs files

## Output
1. Detailed implementation plan
2. `dev/active/[task]/[task]-plan.md`
3. `dev/active/[task]/[task]-context.md`
4. `dev/active/[task]/[task]-tasks.md`
